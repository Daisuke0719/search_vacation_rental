# 札幌 民泊事業 タスク計画

## Phase 1: 調査・準備（1〜2ヶ月目）

---

### 1-1. 法規制の確認（Week 1〜2）

#### A. 住宅宿泊事業法（民泊新法）の調査
- [ ] 住宅宿泊事業法の基本要件を整理
  - 届出制の手続きフロー
  - 年間営業日数上限 180日の計算方法（4/1〜翌3/31）
  - 届出先：札幌市保健所（政令指定都市のため市に届出）
- [ ] 届出に必要な書類一覧の作成
  - 届出書（様式第一）
  - 住宅の図面（各階平面図）
  - 誓約書
  - 登記事項証明書
  - 住民票の写し
  - 消防法令適合通知書
  - 管理規約の写し（マンションの場合）
  - 賃貸借契約書＋転貸承諾書（賃貸の場合）
- [ ] 住宅の定義要件の確認
  - 「台所・浴室・便所・洗面設備」の4設備が必須
  - 「現に人の生活の本拠」or「入居者募集中」or「随時居住可能な状態」

#### B. 札幌市の独自条例の調査
- [ ] 札幌市住宅宿泊事業の適正な運営の確保に関する条例の全文確認
- [ ] 区域制限の確認
  - 住居専用地域での営業制限（曜日・期間制限の有無）
  - 学校周辺での制限の有無
- [ ] 札幌市独自の上乗せ規制の有無
  - 近隣住民への事前周知義務
  - 標識掲示の詳細ルール
  - 苦情対応体制の要件
- [ ] 札幌市保健所への事前相談の予約・実施

#### C. 旅館業法（簡易宿所）との比較検討
- [ ] 簡易宿所営業許可の取得要件を調査
  - 客室面積基準（33㎡以上、10人未満は3.3㎡/人）
  - 換気・採光・照明・防湿・排水の基準
  - フロント設置義務の有無（札幌市の運用確認）
- [ ] 民泊新法 vs 簡易宿所の比較表作成

| 項目 | 民泊新法（届出） | 簡易宿所（許可） |
|------|-----------------|-----------------|
| 営業日数 | 年間180日上限 | 制限なし |
| 手続き | 届出制（比較的簡易） | 許可制（審査あり） |
| 施設基準 | 緩い | 厳しい |
| OTA掲載 | Airbnb等 | Airbnb＋楽天・じゃらん等も可 |
| 初期コスト | 低い | 高い（設備投資） |

- [ ] 特区民泊の適用可否確認（札幌市は対象外の見込みだが最新状況を確認）

#### D. 消防法令の調査
- [ ] 民泊における防火対象物の区分確認
  - 家主居住型：一般住宅扱い（消防設備の追加不要な場合あり）
  - 家主不在型：宿泊施設扱い（自火報、誘導灯等が必要）
- [ ] 必要な消防設備のリストアップ
  - 住宅用火災警報器（全居室・階段）
  - 消火器
  - 誘導灯（避難口・通路）
  - 自動火災報知設備（規模による）
- [ ] 消防法令適合通知書の申請フロー確認
- [ ] 最寄りの消防署への事前相談の予約

#### E. 用途地域・建築基準法の確認
- [ ] 候補エリアの用途地域を札幌市都市計画情報で確認
  - 住居専用地域 → 条例制限の可能性
  - 商業地域・近隣商業地域 → 制限なし
- [ ] 建築基準法上の用途変更の要否確認（200㎡超の場合）
- [ ] 建ぺい率・容積率の確認（購入の場合）

### 1-3. 事業計画策定（Week 3〜6）

#### A. 収支シミュレーション
- [ ] 前提条件の設定
  - 物件タイプ（マンション1R〜2LDK / 戸建て / 一棟）
  - エリア・家賃相場
  - 収容人数
  - 営業形態（民泊新法180日 or 簡易宿所365日）
- [ ] 売上シミュレーション
  - ADR（平均客室単価）の設定：繁忙期 / 通常期 / 閑散期
  - 稼働率の設定：初年度 40〜50%、2年目以降 60〜70%（目標）
  - 年間売上 ＝ ADR × 稼働率 × 営業日数
- [ ] 初期投資の積算

| 項目 | 概算（万円） | 備考 |
|------|------------|------|
| 物件取得費（敷金礼金等） | 30〜60 | 賃貸の場合 |
| 内装リフォーム | 0〜100 | 物件状態による |
| 家具・家電 | 50〜100 | ベッド、TV、冷蔵庫等 |
| 寝具・リネン | 10〜20 | |
| キッチン用品・備品 | 5〜10 | |
| 消防設備 | 5〜30 | 家主不在型の場合 |
| スマートロック | 3〜5 | |
| プロ写真撮影 | 3〜5 | |
| 届出関連費用 | 1〜3 | |
| 予備費 | 10〜20 | |
| **合計** | **120〜350** | |

- [ ] ランニングコストの積算（月額）

| 項目 | 概算（万円/月） | 備考 |
|------|---------------|------|
| 家賃 | 5〜15 | エリア・広さによる |
| 光熱費 | 1〜3 | 冬季は暖房費で増加 |
| Wi-Fi | 0.5 | |
| 清掃費 | 3〜8 | 1回3,000〜5,000円 × 回数 |
| リネンクリーニング | 1〜3 | |
| OTA手数料 | 売上の3〜15% | Airbnb:3%、Booking:15% |
| 管理業者委託費 | 売上の20〜30% | 委託の場合 |
| 消耗品補充 | 0.5〜1 | アメニティ等 |
| 保険 | 0.3〜0.5 | 施設賠償責任保険 |
| 宿泊税 | 変動 | 宿泊料に応じて |
| **合計（自主管理）** | **12〜30** | |

- [ ] 損益分岐点の算出（月間何泊で黒字化するか）
- [ ] 投資回収期間の試算（初期投資 ÷ 月間利益）
- [ ] 3パターンのシナリオ作成（楽観・標準・悲観）

#### B. 営業形態の決定
- [ ] 民泊新法 vs 簡易宿所の収益比較
  - 180日制限の場合の最大売上 vs 365日営業の売上
  - 追加設備投資コストとの損益分岐
- [ ] 判定フロー：
  1. 年間180日で十分な利益が出るか → Yes → 民泊新法（手軽）
  2. No → 簡易宿所の追加投資を回収できるか検討
- [ ] **決定：届出 or 許可の方針確定** ← ★重要マイルストーン

#### C. 運営形態の決定
- [ ] 自主管理と管理委託の比較

| 項目 | 自主管理 | 管理委託 |
|------|---------|---------|
| コスト | 低い | 売上の20〜30% |
| 労力 | 高い（24h対応） | 低い |
| ゲスト対応 | 自分で対応 | 業者が対応 |
| 清掃手配 | 自分で手配 | 業者が手配 |
| スケーラビリティ | 限界あり | 拡大しやすい |

- [ ] 札幌エリアの管理業者リストアップ（3社以上）
- [ ] 管理業者への問い合わせ・見積もり取得
- [ ] **決定：自主管理 or 管理委託の方針確定** ← ★重要マイルストーン

#### D. 資金調達計画
- [ ] 必要資金の総額確定（初期投資＋運転資金3〜6ヶ月分）
- [ ] 資金調達方法の検討
  - 自己資金
  - 日本政策金融公庫（創業融資）
  - 民間銀行ローン
  - 補助金・助成金の有無確認（札幌市、北海道）
- [ ] 事業計画書のドラフト作成（融資申請用）

- [ ] 消防署への事前相談（必要設備の確認）

## 札幌特有の注意事項

| 項目 | 内容 |
|------|------|
| **冬季対策** | 暖房設備は必須。水道凍結防止（水落とし手順のマニュアル化）。除雪対応。 |
| **インバウンド需要** | ニセコ・富良野等と組み合わせた広域観光需要。多言語対応が重要。 |
| **繁忙期** | 2月（雪まつり）、7-8月（夏季観光）、12-3月（スキーシーズン） |
| **閑散期** | 4-5月、10-11月 → 価格調整やビジネス利用の取り込みが鍵 |
| **宿泊税** | 札幌市は宿泊税を導入予定/導入済 → 最新状況を確認 |
| **交通** | 地下鉄駅徒歩圏内が望ましい。新千歳空港からのアクセスも訴求ポイント |
