## 1. 住宅宿泊事業法（民泊新法）の要件

**根拠法令**: 住宅宿泊事業法（平成29年法律第65号）、2018年6月15日施行
**参照**: https://www.mlit.go.jp/kankocho/minpaku/overview/minpaku/law1.html

### 1.1 届出手続きのフロー

住宅宿泊事業は「届出制」であり、都道府県知事等（札幌市の場合は札幌市長）への届出が必要。

```
【届出手続きの流れ】

① 事前確認
   ├─ 用途地域の確認（都市計画課）
   ├─ 建築基準法上の適合確認
   ├─ マンションの場合：管理規約で民泊が禁止されていないか確認
   └─ 賃貸物件の場合：賃貸人（オーナー）の転貸承諾を取得

② 消防署への事前相談・消防法令適合通知書の取得
   ├─ 管轄消防署に事前相談
   ├─ 必要な消防設備の設置
   └─ 消防法令適合通知書の交付申請・取得

③ 札幌市条例に基づく事前手続き
   ├─ 近隣住民への事前周知（後述）
   └─ 制限区域の該当有無確認

④ 届出書類の準備・提出
   ├─ 民泊制度運営システム（オンライン）での届出が原則
   └─ 必要書類一式を添付

⑤ 届出受理・届出番号の発行
   ├─ 受理までの期間：概ね1〜3週間
   └─ 届出番号が民泊制度ポータルサイトで公表

⑥ 仲介サイト（Airbnb等）への届出番号登録・営業開始
```

**届出先**: 札幌市経済観光局観光・MICE推進部
**参照**: https://www.city.sapporo.jp/keizai/kanko/minpaku/20190409after.html

### 1.2 必要書類一覧

**根拠**: 住宅宿泊事業法施行規則第4条
**参照**: https://www.mlit.go.jp/kankocho/minpaku/business/host/procedure_doc.html

| No. | 書類名 | 備考 |
|-----|--------|------|
| 1 | 住宅宿泊事業届出書（様式第1号） | 民泊制度運営システムで作成 |
| 2 | 住宅の図面 | 各設備の位置、間取り、入口、階、居室・宿泊室・宿泊者使用部分の床面積を記載 |
| 3 | 住宅の登記事項証明書 | 未登記の場合は固定資産税課税台帳の写し等 |
| 4 | 住宅が「入居者の募集が行われている家屋」に該当する場合の証明書類 | 賃貸募集広告等 |
| 5 | 「随時居住の用に供されている家屋」に該当する場合の証明書類 | 該当する場合のみ |
| 6 | 転貸承諾書 | 賃借人が届出する場合（賃貸人の承諾） |
| 7 | マンション管理規約の写し | 管理規約で民泊が禁止されていないことの確認 |
| 8 | 管理規約に民泊に関する定めがない場合 | 管理組合に禁止する意思がないことを確認した書類 |
| 9 | 消防法令適合通知書 | 管轄消防署で取得 |
| 10 | 欠格事由に該当しないことの誓約書 | 法第4条各号の欠格事由 |
| 11 | 住民票の写し（個人の場合） | 届出日前3月以内発行のもの |
| 12 | 登記事項証明書・定款（法人の場合） | 法人の場合 |
| 13 | 役員の破産手続開始決定を受けて復権を得ない者でないことの証明書 | 法人の場合、市区町村長の証明 |
| 14 | 住宅宿泊管理業者への管理委託契約書の写し | 家主不在型の場合は必須 |

> **注意**: 官公署が証明する書類は届出日前3月以内に発行されたもの。書類は日本語又は英語で記載（英語の場合は日本語翻訳文を添付）。

### 1.3 住宅の定義要件（4設備等）

**根拠**: 住宅宿泊事業法第2条第1項

住宅宿泊事業法における「住宅」とは、以下の**4つの設備**がすべて備わっていることが必要：

| 設備 | 要件 |
|------|------|
| **台所** | 調理のための設備（コンロ・流し台等） |
| **浴室** | 入浴またはシャワーのための設備 |
| **便所** | トイレ |
| **洗面設備** | 洗面台等 |

これら4設備が「設けられている」ことが住宅の最小単位の要件。

さらに、以下のいずれかに該当する家屋であること：

1. **現に人の生活の本拠として使用されている家屋**（居住中の自宅等）
2. **入居者の募集が行われている家屋**（賃貸募集中の空き家等）
3. **随時その所有者、賃借人又は転借人の居住の用に供されている家屋**（別荘・セカンドハウス等、年1回以上使用実績があるもの）

### 1.4 年間180日の計算方法

**根拠**: 住宅宿泊事業法第2条第3項、同法施行規則第3条

- **計算期間**: 毎年**4月1日正午**から翌年**4月1日正午**まで（暦年ではない）
- **上限日数**: 年間**180日**（届出住宅ごとにカウント）
- **1日のカウント方法**: **正午（12:00）から翌日正午（12:00）**を1日とする
  - 例：4月1日15:00チェックイン → 4月3日10:00チェックアウト = **2日**
  - 例：4月1日15:00チェックイン → 4月3日13:00チェックアウト = **3日**（4月3日正午時点で宿泊者が滞在しているため）
- **届出住宅単位**: 事業者単位ではなく、届出住宅ごとにカウント（複数物件運営の場合はそれぞれ180日）
- **事業者の変更**: 年度途中で住宅宿泊事業者が変わっても、その住宅の180日カウントは引き継がれる
- **条例による制限**: 自治体の条例でさらに短縮される場合がある（札幌市の場合は後述）

**参照**: https://tabipa-ma.com/blog/180-day-rule/

## 2. 札幌市の独自条例

### 2.1 札幌市住宅宿泊事業の実施の制限に関する条例

**根拠**: 札幌市住宅宿泊事業の実施の制限に関する条例（平成30年札幌市条例第8号）
**参照**: https://www.city.sapporo.jp/keizai/kanko/news2/seigenkuiki.html

住宅宿泊事業法第18条に基づき、札幌市が独自に定めた上乗せ規制。

### 2.2 区域制限（制限区域）

以下の区域において、住宅宿泊事業の実施が制限される：

#### 制限区域の種類

| 制限区域 | 内容 |
|----------|------|
| **住居専用地域** | 第一種低層住居専用地域、第二種低層住居専用地域、第一種中高層住居専用地域、第二種中高層住居専用地域 |
| **学校周辺** | 小学校・中学校等の敷地の出入口（正門等）の周囲**100m**の範囲 |

#### 制限の対象者

以下の**いずれか**に該当する場合に制限が適用される：

1. 届出住宅を**自己の生活の本拠として使用していない**もの
2. 届出住宅に人を宿泊させる間、住宅宿泊事業者が**不在となる**もの（家主不在型）
3. 届出住宅の**居室の数が5を超える**もの

> **つまり**: 家主居住型で居室5室以下の場合は、制限区域内でも制限を受けない。

#### 制限される期間

制限区域内では、以下の期間**のみ**営業可能：

- **金曜日正午 〜 月曜日正午**（週末）
- **祝前日正午 〜 祝翌日正午**（祝日前後）

上記以外の平日（月曜日正午〜金曜日正午）は営業不可。

> **影響**: 実質的に営業可能日数は年間約**120〜140日程度**に制限される可能性がある。

### 2.3 近隣住民への事前周知義務

**参照**: https://www.city.sapporo.jp/keizai/kanko/minpaku/documents/minpakutebiki20230406.pdf

札幌市の手引きにおいて、住宅宿泊事業を開始する前に以下の対応が求められる：

- 近隣住民に対して、民泊事業を営む旨を**事前に説明**すること
- **緊急時の連絡先**を周知すること
- 近隣住民との**トラブル防止**のための対応

> 札幌市では、届出前に近隣住民への挨拶・説明を行い、連絡先を伝えることが推奨されている。

### 2.4 上乗せ規制のまとめ

| 項目 | 国の法律（民泊新法） | 札幌市条例（上乗せ） |
|------|----------------------|----------------------|
| 営業日数上限 | 年間180日 | 制限区域内は週末・祝日のみ（実質120〜140日程度） |
| 区域制限 | なし（住宅が建てられる用途地域） | 住居専用地域・学校周辺100mで制限 |
| 近隣周知 | 法律上の明示的義務なし | 事前説明が求められる |
| 制限の適用除外 | ― | 家主居住型かつ居室5室以下 |

**相談窓口**: 札幌市民泊総合窓口
所在地：札幌市中央区北1条西2丁目2-1 北海道経済センタービル7階

## 3. 旅館業法（簡易宿所）との比較

### 3.1 簡易宿所営業許可の取得要件

**根拠**: 旅館業法（昭和23年法律第138号）第3条
**参照**: https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000141689.html

簡易宿所営業とは、宿泊する場所を多数人で共用する構造及び設備を設けてする営業（カプセルホテル、ゲストハウス、民宿等）。

#### 許可取得の流れ

```
① 事前相談（保健所・消防署・建築指導課）
② 用途地域の確認
③ 建築基準法上の用途変更（必要な場合）
④ 消防設備の設置・消防法令適合通知書の取得
⑤ 施設の整備（施設基準の充足）
⑥ 営業許可申請（保健所へ）
⑦ 施設検査（保健所職員による立入検査）
⑧ 許可証の交付
⑨ 営業開始
```

#### 主な許可要件

- 欠格事由に該当しないこと
- 施設基準を満たすこと
- 用途地域の制限に適合すること
- 衛生基準を満たすこと

### 3.2 施設基準（面積・設備等）

| 項目 | 基準 |
|------|------|
| **客室延床面積** | 合計 **33㎡以上**（宿泊者数10人未満の場合は1人あたり3.3㎡以上） |
| **1人あたり床面積** | **3.3㎡以上** |
| **採光・換気** | 窓の面積は床面積の **1/7以上** |
| **照明設備** | 適切な照明の設置 |
| **浴室・シャワー** | 宿泊者の需要に応じた数の浴室又はシャワー室 |
| **トイレ** | 適切な数のトイレ設備 |
| **換気設備** | 適切な換気ができること |
| **玄関帳場（フロント）** | 原則必要（ただし代替措置が認められる自治体あり） |

### 3.3 民泊新法 vs 簡易宿所 比較表

| 比較項目 | 民泊新法（住宅宿泊事業法） | 簡易宿所（旅館業法） |
|----------|---------------------------|---------------------|
| **制度の種類** | 届出制 | 許可制 |
| **営業日数** | 年間**180日以内** | **制限なし**（365日営業可能） |
| **手続きの難易度** | 比較的容易 | やや煩雑（施設検査あり） |
| **施設基準** | 4設備（台所・浴室・便所・洗面）があれば可 | 客室面積33㎡以上、採光基準等あり |
| **用途地域** | 住宅が建てられる地域ならどこでも | ホテル・旅館が建てられる用途地域のみ |
| **建築確認（用途変更）** | **不要**（住宅のまま） | 200㎡超の場合**必要** |
| **消防設備** | 家主居住型（50㎡以下）は住宅扱い | 旅館・ホテル並みの設備が必要 |
| **管理義務** | 家主不在型は管理業者委託が必須 | 管理者の常駐が原則 |
| **フロント（帳場）** | **不要** | **原則必要** |
| **初期費用目安** | 数万円〜30万円程度 | 数十万円〜数百万円 |
| **収益性** | 180日制限のため限定的 | 365日営業可能で高い |
| **自治体上乗せ規制** | あり（条例で日数等がさらに制限） | 自治体の条例による |

> **判断基準**: 年間稼働率が50%（約180日）以内で十分な場合は民泊新法、通年営業で収益最大化を図る場合は簡易宿所が適している。

## 4. 消防法令の要件

**根拠**: 消防法（昭和23年法律第186号）、消防法施行令
**参照**: https://www.fdma.go.jp/mission/prevention/suisin/items/minpaku_leaf_horei.pdf

### 4.1 家主居住型 vs 家主不在型での消防設備の違い

民泊の消防法上の取扱いは、家主の居住状況と宿泊室の面積により異なる。

#### 判定フロー

```
家主が不在とならないか？
  ├── YES（家主居住型）
  │     └── 宿泊室の床面積合計が50㎡以下か？
  │           ├── YES → 「一般住宅」扱い（住宅用火災警報器のみ）
  │           └── NO  → 「宿泊施設（5項イ）」扱い（消防設備必要）
  └── NO（家主不在型）
        └── すべて「宿泊施設（5項イ）」扱い（消防設備必要）
```

| 類型 | 消防法上の扱い | 必要な消防設備 |
|------|---------------|---------------|
| **家主居住型・宿泊室50㎡以下** | 一般住宅 | 住宅用火災警報器のみ |
| **家主居住型・宿泊室50㎡超** | 5項イ（旅館・ホテル等） | 下記の消防設備一式 |
| **家主不在型（面積問わず）** | 5項イ（旅館・ホテル等） | 下記の消防設備一式 |

### 4.2 消防法令適合通知書の申請フロー

```
① 管轄消防署への事前相談
   └─ 届出住宅の図面等を持参し、必要な消防設備を確認

② 消防設備の設置工事
   └─ 自動火災報知設備、誘導灯等の設置

③ 消防法令適合通知書交付申請
   └─ 申請書と必要書類を消防署に提出

④ 消防署の立入検査
   └─ 消防職員が現地を検査

⑤ 消防法令適合通知書の交付
   └─ 検査合格後に交付（通常1〜2週間程度）
```

**札幌市の消防相談先**: 各区の消防署予防課
**参照**: https://www.city.sapporo.jp/shobo/yobo/shido/minpaku.html

### 4.3 必要な消防設備リスト（5項イに該当する場合）

| 消防設備 | 設置基準 | 備考 |
|----------|----------|------|
| **自動火災報知設備** | 延べ面積に関係なく設置義務あり | 2階建以下・延べ面積300㎡未満の一戸建てでは「**特定小規模施設用自動火災報知設備**」（無線式）で可 |
| **誘導灯** | 避難口・通路に設置 | 避難口誘導灯（玄関等の避難口直上）、通路誘導灯（廊下の曲がり角等） |
| **消火器** | 延べ面積150㎡以上（地階・無窓階・3階以上は50㎡以上） | 面積未満の場合は義務なし（設置推奨） |
| **防炎物品の使用** | カーテン、じゅうたん等 | 防炎ラベルのあるものを使用 |
| **スプリンクラー設備** | 11階以上の階 | 高層マンション等の場合 |
| **住宅用火災警報器** | 全住宅に義務 | 5項イでも住警器を自火報の代替にはできない |

#### 費用目安

- **特定小規模施設用自動火災報知設備**: 約10〜20万円（無線式、配線工事不要）
- **誘導灯**: 約5〜10万円
- **消火器**: 約5,000〜1万円/本
- **合計目安（一戸建て）**: 約**20〜35万円程度**

### 5.2 建築基準法上の用途変更の要否

**根拠**: 建築基準法第87条
**参照**: https://minpaku.yokozeki.net/youtohenkou/

#### 民泊新法（住宅宿泊事業法）の場合

- **用途変更は不要**
- 住宅宿泊事業法に基づく民泊は、建築基準法上の用途は「住宅」のまま
- 建築確認申請は不要

#### 簡易宿所（旅館業法）の場合

- 元の建物用途が「住宅」等から「旅館・ホテル」に変わるため、**用途変更が必要**となる場合がある
- **用途変更の確認申請が必要な場合**: 旅館業に供する部分の面積が**200㎡を超える**とき
- **用途変更の確認申請が不要な場合**: 旅館業に供する部分の面積が**200㎡以下**のとき
  - ただし、用途変更の確認申請が不要でも、建築基準法の各規定（防火、避難等）への適合は必要

> **注意**: 2019年の建築基準法改正により、用途変更の確認申請が必要な面積の基準が100㎡超から200㎡超に緩和された。

## 6. 特区民泊の適用可否

**根拠**: 国家戦略特別区域法（平成25年法律第107号）第13条
**参照**: https://www.mlit.go.jp/kankocho/minpaku/overview/minpaku/law3.html

### 結論: 札幌市は特区民泊の対象外

札幌市は**国家戦略特別区域に指定されていない**ため、特区民泊（国家戦略特別区域外国人滞在施設経営事業）は利用できない。

**参照**: https://www.city.sapporo.jp/kikaku/kokkasennryakutokku/kokkasennryakutokku.html

| 千葉県千葉市 | |

#### 特区民泊との比較（参考）

| 項目 | 特区民泊 | 民泊新法 |
|------|---------|---------|
| 営業日数 | 制限なし（365日可能） | 180日以内 |

## 7. 宿泊税について

### 7.1 札幌市宿泊税の概要

**根拠**: 札幌市宿泊税条例
**参照**: https://www.city.sapporo.jp/citytax/shukuhakuzei/index.html

#### 導入時期

- **2026年（令和8年）4月1日宿泊分から課税開始**
- 2025年3月21日に総務大臣の同意を取得

#### 課税対象

- 札幌市内の**旅館・ホテル・簡易宿所・住宅宿泊事業（民泊）** 等の宿泊施設に宿泊する者
- **民泊事業者も宿泊税の特別徴収義務者**となる

#### 税率（北海道宿泊税との合算）

札幌市では、北海道宿泊税と札幌市宿泊税が**同時に課税**される（二重課税ではなく、道と市がそれぞれ課税）。

| 1人1泊あたりの宿泊料金 | 北海道宿泊税 | 札幌市宿泊税 | **合計** |
|------------------------|:-----------:|:-----------:|:-------:|
| **20,000円未満** | 100円 | 200円 | **300円** |
| **20,000円以上 50,000円未満** | 200円 | 200円 | **400円** |
| **50,000円以上** | 500円 | 500円 | **1,000円** |

#### 免税対象

以下に該当する宿泊者は課税免除：

- 学校教育法第1条に規定する学校（大学を除く）が主催する**修学旅行その他学校行事**に参加している幼児・児童・生徒・学生及び引率者
- 認定こども園・保育所等が主催する行事に参加している**満3歳以上の幼児及び引率者**

#### 民泊事業者への影響

- 民泊事業者は宿泊税の**特別徴収義務者**として、宿泊者から宿泊税を徴収し、札幌市に申告・納付する義務がある
- 宿泊料金の設定時に宿泊税分を考慮する必要がある
- **宿泊税システムの整備費用に対する補助金制度**あり（札幌市宿泊税システム整備費補助金）
  - 参照: https://www.city.sapporo.jp/citytax/shukuhakuzei/shisutemuseibihihojokin.html
